Group Project: Create a Website Using WIX

This project was created in my Human-Computer Interaction course. The focus was to create any website using a resource available online with purpose of solving a problem or creating a service. My group's idea was to create a website that graphic designers can use to display their art (portfolio) and connect with other graphic designers on the website.

What I Learned

This project challenged my leadership and communication skills. All group members came from different backgrounds and had conflicting schedules to work with. With great communication and initiative from all parties, we were able to complete the project and meet our design expectations.
